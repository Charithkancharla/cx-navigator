# Telephony Backend Setup Guide

To use the live IVR discovery features (calling real phone numbers), you must run the separate Node.js telephony server and expose it to the internet so Convex can communicate with it.

## 1. Prerequisites
- Node.js installed
- A Twilio Account (SID, Auth Token, and a Phone Number)
- A Deepgram API Key
